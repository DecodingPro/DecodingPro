# Security Policy

## Supported Versions
Only the **latest version** of the free tier is supported.  
Premium and Core vaults receive security updates for 12 months after purchase.

## Reporting a Vulnerability

**Do NOT open public issues.**  
Please email `security@dp.com` with:

- A clear description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact

We will:
- Acknowledge receipt within **48 hours**
- Provide a timeline for a fix
- Credit you in release notes (if desired)

We follow industry‑standard disclosure practices and take security seriously – even for free code.

## Scope
This policy applies to all code in this repository (free tier).  
For vulnerabilities in paid vaults, please use the same email address.
